Attributes used: controls, loop, multiple source formats

Key Attributes Learned

  • controls - Displays play/pause/volume controls
  • autoplay - Starts media automatically - use carefully!
  • loop - Repeats media when finished
  • muted - Starts with audio muted
  • poster - Thumbnail image before video plays
  • preload - Hints how much to buffer: none/metadata/auto
